# Now Playing

A lightweight web component that displays my currently playing (or recently played) track from Last.fm.

## Features

- Shows track title, artist, and album art
- Displays "Listening now" if a track is actively playing
- Falls back to last played track with timestamp

## Stack

- HTML/CSS/JS
- Netlify Functions (serverless)
- Last.fm API
